Perspective Projection simulates depth.

Perspective Projection Matrix (4×4)

f = 1 / tan(fov/2) | n = near, fa = far

P = [
  [f/aspect, 0, 0, 0 ],
  [0, f, 0, 0 ],
  [0, 0, (fa+n)/(n-fa), (2*fa*n)/(n-fa)],
  [0, 0, -1, 0 ]
]

Homogeneous divide: x' = x/w, y' = y/w, z' = z/w
